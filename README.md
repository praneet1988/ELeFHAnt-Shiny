# ELeFHAnt-Shiny
ELeFHAnt: A supervised machine learning approach for label harmonization and annotation of single cell RNA-seq data (Shiny App)

# Launch App
```
library(shiny)
runGitHub("ELeFHAnt-Shiny", "praneet1988", launch.browser = TRUE, ref="main")

```
